# HRD-Viewer für Jugendforscht
Ein Program welches ein HRD-Diagram zeigt für ein Jugendforscht-Projekt vom Astro-Club
beim Physikalischen Verein Frankfurt am Main.

Es benutzt Python als Programmiersprache. Für die GUI wird PySide6 (also QT6) verwendet
und es wird an matplotlib angebunden um Graphen anzuzeigen.
